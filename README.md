# 🤖 AI Chatbot Project

This is my first AI chatbot project built with HTML, CSS, and JavaScript.

## Features
- Natural language responses
- Tells jokes
- Shows current time and date
- Help system
- Clean chat interface

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Git for version control

## How to Use
1. Open `index.html` in a browser
2. Type: hello, time, date, joke, or help
3. Press Enter or click Send
