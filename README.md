# 🤖 Bilingual AI Chatbot (English/العربية)

**🌐 Live Demo:** [https://talkyw-cyber.github.io/ai-chatbot/](https://talkyw-cyber.github.io/ai-chatbot/)

This is a bilingual AI chatbot that understands and responds in both English and Arabic.

## 🌟 Features
- **Bilingual Conversations:** Communicate in English or Arabic (or mix both!)
- **Smart Language Detection:** Automatically detects which language you're using
- **Multiple Response Types:**
  - Greetings (hello, مرحبا)
  - Current time and date (time, الوقت, date, التاريخ)
  - Programming jokes (joke, نكتة)
  - Help instructions (help, مساعدة)
- **Modern UI:**
  - Responsive design
  - Language selector (English/Arabic/Both)
  - RTL (Right-to-Left) support for Arabic
  - Smooth animations and gradient backgrounds

## 🛠️ Technologies Used
- HTML5 (with UTF-8 encoding for Arabic support)
- CSS3 (with RTL styling and modern gradients)
- JavaScript (ES6 with language detection algorithms)
- Git & GitHub (for version control and hosting)

## 🚀 How to Use

### English:
1. Open the [live demo](https://talkyw-cyber.github.io/ai-chatbot/)
2. Type commands like: hello, time, date, joke, help
3. Press Enter or click Send

### العربية:
1. افتح [التجربة الحية](https://talkyw-cyber.github.io/ai-chatbot/)
2. اكتب أوامر مثل: مرحبا, الوقت, التاريخ, نكتة, مساعدة
3. اضغط Enter أو انقر على إرسال

### Mixed Language:
You can mix English and Arabic in the same message!
